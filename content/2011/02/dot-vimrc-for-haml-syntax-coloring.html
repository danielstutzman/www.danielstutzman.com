--- layout: post title: dot vimrc for HAML syntax coloring date: 2011-02-21 ---

Vim under Ubuntu Hardy apparently doesn't come preinstalled with syntax coloring for .haml files.  To add this, follow stack overflow's instructions.  If it still doesn't work, try adding an extra "filetype off" line as follows:

set runtimepath+=~/.vim/vim-haml
filetype off
filetype plugin indent on
syntax on
